# Hubble Pad

A powerful work management tool for organizing and tracking your work items.

## Installation

[npm package](https://www.npmjs.com/package/hubble-pad)

```bash
# Install the latest stable version
npm install -g hubble-pad

# Or install a specific beta version
npm install -g hubble-pad@beta
## Usage

```bash
# Start the server (default port: 10002)
hubble-pad start

# Check version
hubble-pad --version
```

## Configuration

The server runs on port 10002 by default. You can change it by setting the `PORT` environment variable:

```bash
PORT=8080 hubble-pad start
```
